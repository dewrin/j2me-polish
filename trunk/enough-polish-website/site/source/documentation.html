Installation

Installation is easy. <%ifdef excludeDownload %> Start the installer by either double-clicking it or calling "java -jar j2mepolish-<%= PolishVersion %>.jar". <%else %> Just download the latest distribution and start the installation by either double-clicking it or calling "java -jar j2mepolish-<%= PolishVersion %>.jar". <%endif %> This is described in more detail in the installation documentation.

build.xml / J2ME Polish Task

The build.xml file controls the build process. The possible attributes and the definition of the Ant task can be found at the build documentation.

CSS Design for J2ME

You can design your application using CSS textfiles. This technique is explained in detail at the CSS documentation.

Preprocessing

You can optimize your application to different devices very easily without loosing the portability of your application. This is explained in the preprocessing documentation.

Utilities

J2ME Polish includes some often needed classes like the ArrayList or the Debug class. They are explained in the utilities documentation.
